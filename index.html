To design two button

２つのデザインを用意します。

  1. touchしたくなるようなデザイン
  2. touchされてヘコんでるデザイン

HTML

HTMLは "button" というclass名だけの要素と、"button active"という2つのclass名を併記した要素をmarkupしてましょう。
jQuery本体とtouchclick.jsを読み込むのも忘れずに。

<script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/jquery/1.7.2/jquery.min.js"></script>
<script type="text/javascript" src="../js/jquery.touchclick.js"></script>

<a class="button">button</a>
<a class="button active">button</a>

CSS

"button"というclass名を基準に、"button active"のstyleも作成します。

.button{
    display:inline-block;
    padding:0.5em 1em;
    border:1px solid #bbb;
    border-radius:1em;
    background:
      -webkit-gradient(
        linear,0 0,0  100%,
        from(#fff),
        to(#ddd));
    -webkit-box-shadow:
      0 1px 1px #bbb,0 3px 1px #ddd;
    box-shadow:
      0 1px 1px #bbb,0 3px 1px #ddd;
    font-weight:bold;
}
.button.active{
    -webkit-transform:translate(0,2px);
    background:
      -webkit-gradient(
        linear,0 0,0 100%,
        from(#eee),to(#ccc));
    -webkit-box-shadow:
      0 1px 1px #bbb,0 1px 1px #ddd;
    box-shadow:
      0 1px 1px #bbb,0 1px 1px #ddd;
}

JavaScript

ここまでくればあとは簡単。
touchClickプラグインを適用する JavaScript Code はこれです。

$('.button').touchClick(function (event) {
  // your code.
});
